# SRM-INSIDER
# 🚀 SRM Insider

A modern, responsive student platform designed to share insights on placements, internships, and campus life.

## 🌐 Live Demo

🔗 https://srminsider3.netlify.app/

---

## 🧠 About the Project

SRM Insider is a frontend web application focused on delivering a smooth and engaging user experience for students. It provides curated content related to placements, internships, and campus experiences.

The project emphasizes clean UI design, smooth animations, and intuitive navigation.

---

## ✨ Features

* 🎨 Modern and responsive UI
* 🌌 Animated hero section (Aurora + grid effect)
* 📊 Scroll progress indicator
* 🔍 Search and filter functionality
* 📄 Dynamic content rendering (articles/posts)
* 🧩 Interactive dashboard layout
* ✨ Smooth hover effects and transitions

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## ⚙️ How It Works

* Content is managed using a JavaScript data structure (POSTS array)
* Cards and articles are dynamically rendered using DOM manipulation
* Smooth UI interactions are implemented using CSS transitions and animations

---

## 📱 Responsiveness

The website is fully responsive and works across:

* Desktop
* Tablet
* Mobile devices
